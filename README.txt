
# Nutrition Agent using IBM WatsonX

This project was developed as part of an internship program with **IBM** and the **Edunet Foundation**.
It utilizes IBM WatsonX to build an AI-powered Nutrition Agent that provides daily meal suggestions
based on balanced diet principles.

## Features
- Built using IBM WatsonX's Agent Lab
- Provides healthy meal suggestions (breakfast, lunch, dinner, snacks)
- Simple and user-friendly interface
- Hosted and tested on IBM Cloud
